Extract mapping CSV(s) from enriched d365-entities/ JSON files.

This is Step 2 of the pipeline. Preserves confirmed SF mapping columns (sfObjectName, sfFieldDisplayName, sfFieldApiName) from existing CSVs. Clears sfSuggested* columns (regenerated by step 4).

If `$ARGUMENTS` is empty, run for all entities. Otherwise run for the specified entity.

```bash
# Determine argument
if [ -z "$ARGUMENTS" ]; then
  ARG="--all"
else
  ARG="$ARGUMENTS"
fi

python scripts/generate_d365_entity_csv_mapping_step_02.py $ARG
```

After completion, report:
- Number of mapping CSVs generated (count files in mapping/)
- Any warnings or errors from the run
